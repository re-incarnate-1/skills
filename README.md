# Claude Agent Skills

A collection of custom skills for Claude, designed to enhance behavior and capabilities.

## Available Skills

### 📂 [Honesty](./honesty/SKILL.md)
A protocol for radical transparency. This skill instructs Claude to admit uncertainty, avoid people-pleasing fabrications, and proactively offer speculation only when confidence is high.

## How to install Honesty
If you are using Claude Code or a compatible agent, you can install these skills directly from this repository:

npx skills@latest add re-incarnate-1/skills/honesty

## 📂 [Examined Thinking](examined-thinking/README.md)
This skill structures how Claude thinks with you rather than just at you—answering directly, then questioning and reexamining where it matters. It avoids the need to manually ask for suggestions or follow-up questions in every prompt.

### How to install Examined-Thinking
`npx skills@latest add re-incarnate-1/skills/examined-thinking`
